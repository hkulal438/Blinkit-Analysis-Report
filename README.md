**Blinkit Dashboard**
Blinkit Dashboard is an analytics solution designed for India's Last Minute App. The dashboard visualizes sales performance, outlet details, and product metrics, helping business owners and analysts make informed decisions.

Features
Sales Metrics: Track total sales, average sales per item, and total number of items sold.

Outlet Insights: Visualize establishments across years, see breakdowns by outlet size, location, and type.

Product Analysis: View sales performance by item type and fat content, with easy comparison across multiple product categories.

Filtering: Apply filters for Outlet Location Type, Outlet Size, and Item Type for targeted analyses.
